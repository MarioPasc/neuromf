# Papers Reference Library

This directory contains PDFs and structured insight documents for all papers relevant to the NeuroMF project.

## Structure

Each paper has its own folder:
```
docs/papers/{name}_{year}/
├── {paper}.pdf          # Paper PDF (placed manually)
├── README.md            # Paper metadata and instructions
└── insights.md          # Generated by /review-external (structured analysis)
```

## Workflow

1. Place the paper PDF in the appropriate folder.
2. Run `/review-external <paper_folder> <external_repo>` to generate `insights.md`.
3. The phase-implementer subagent reads `insights.md` during implementation.
